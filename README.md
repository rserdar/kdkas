# KDK Portal

Modern Kayıt ve Denetim Platformu.

## 🚀 Teknolojiler

- **Core:** Astro 6.x
- **Styling:** Tailwind CSS 4.x (Pure Tailwind)
- **Design:** Modern Premium UI with Dark Mode & Glassmorphism
- **Typography:** Inter & Outfit (Google Fonts)

## 🛠️ Kurulum

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

## 🏗️ Mimari

Proje, Astro'nun modern ada mimarisini ve Tailwind 4'ün performanslı yapısını kullanmaktadır. Tüm UI bileşenleri `src/styles/global.css` dosyasındaki merkezi tasarım sistemi üzerine inşa edilmiştir.

## 🎨 Tasarım Prensipleri

1. **Rich Aesthetics:** Derin karanlık mod, gradiyentler ve cam efekti (glassmorphism).
2. **Visual Excellence:** Modern tipografi ve yumuşak mikro-animasyonlar.
3. **Dynamic Design:** Hover efektleri ve akıcı geçişler.
